## 💡 Pro Tips

### 💾 Save Your JWT Token
After login, save the token to a file:
```bash
# In terminal
export JWT_TOKEN="token_from_response"

# Or with jq
JWT=$(curl -s ... | jq -r '.data.token')
```

### 📌 Postman Variable Tricks
Set variables from responses automatically:
```javascript
// In Tests tab
var token = pm.response.json().data.token;
pm.environment.set("jwt_token", token);
```

### 🔄 Test in Sequence
Always test in this order:
1. Auth (register/login)
2. Create resources
3. Read resources
4. Update resources
5. Delete resources

### 🗄️ Check Database Console
Run along with testing:
```bash
npx prisma studio
# Opens at http://localhost:5555
```
